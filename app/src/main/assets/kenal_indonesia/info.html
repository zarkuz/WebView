Aplikasi ini merupakan aplikasi berbasis android yang dikembangkan dengan bahasa pemrograman java, html, dan xml. Selain itu aplikasi ini juga menggunakan framework bootsrap dan script yang diambil lewat forum bootlip. Aplikasi ini merupakan aplikasi local web view atau web yang kemudian dibuat apk namun bersifat local atau file web berada dalam aplikasi tersebut (tidak dihosting). Aplikasi ini bisa diakses dalam keadaan online maupun offline namun untuk melihat video dan memutar lagu harus terkoneksi dengan internet karena file lagu dan internet disimpan di internet untuk memperkecil ukuran aplikasi tersebut. Untuk video disimpan pada laman youtube.com yang diambil dari beberapa pengguna yang berbeda. Untuk lagu disimpan pada hosting pribadi pembuat dengna memanfaatkan fasilitas hosting gratis dari idhostinger. Untuk informasi yang ada dalam aplikasi diambil dari beberapa blog yang ada internet serta informasi dari laman wikipedia. Untuk kritik dan saran dapat disampaikan lewat pada kolom komentar pada laman download aplikasi.
